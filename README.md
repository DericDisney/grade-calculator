# grade-calculator
Python  app to calculate student grades and GPA based on subject marks and credits.

## Features
- Input any number of subjects with marks and credit hours
- Automatically calculates grade letter (A+, A, B+...) and grade points
- Computes overall GPA
- Identifies failed subjects and displays pass/fail status
- Clean formatted result table

## How to Run
1. Make sure Python is installed on your system
2. Download or clone this repository
3. Run the following command:

python grade_calculator.py

## Technologies Used
- Python 3
- Object-Oriented Programming (OOP)
- Input validation and error handling
